# Vindsiden frontend

##  How to run locally

1. [Download node](https://nodejs.org/en/download/)
2. Install packages with ``npm install``
3. Start node server with ``npm run dev``
4. Open [http://localhost:4000/](http://localhost:4000/) in your browser
